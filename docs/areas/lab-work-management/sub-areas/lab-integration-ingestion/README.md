# Lab Integration & Result Ingestion

Connectors and normalization for lab results, attachments, and provider integrations.

## Functions

| Function | Description |
|----------|-------------|
| hl7-fhir-lab-result-ingest | Ingest HL7 v2 ORU and FHIR DiagnosticReport messages |
| lis-api-connector | Vendor-specific LIS connectors and polling strategies |
| result-parsing-validation | Parse attachments and validate observations |
| result-versioning | Manage versions of results and attachments |
