# Order Management

This sub-area covers creating, editing, routing, and tracking lab orders.

## Functions

| Function | Description |
|----------|-------------|
| create-lab-order | Create a new lab order for a patient |
| order-status-tracking | Track statuses and events for orders |
| order-editing-cancellation | Edit or cancel an existing order |
| order-priority-and-routing | Set priority and route to appropriate lab provider |

Function specs are in `functions/`.
