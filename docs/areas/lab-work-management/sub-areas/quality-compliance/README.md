# Quality & Compliance

Audit, retention, consent, and QA processes for lab workflows.

## Functions

| Function | Description |
|----------|-------------|
| qa-audit-logging | Ensure all lab events are logged for audit |
| retention-policy-enforcement | Enforce configurable retention and archival |
| consent-and-privacy-controls | Track consent for specimen and report use |
| proficiency-test-tracking | Track lab QA/proficiency testing events |
