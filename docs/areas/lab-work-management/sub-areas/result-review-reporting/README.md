# Result Review & Reporting

Clinician review, sign-off, patient notifications and report generation.

## Functions

| Function | Description |
|----------|-------------|
| clinician-review-and-signoff | Review UI and sign-off workflow for results |
| automated-abnormal-flagging | AI-assisted abnormal result detection |
| patient-notification-delivery | Configure and send result notifications |
| lab-report-generation | Generate consolidated lab reports (PDF) |
