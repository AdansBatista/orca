# Sample Collection & Tracking

Specimen labeling, collection workflows, chain-of-custody, and location tracking.

## Functions

| Function | Description |
|----------|-------------|
| specimen-labeling-barcode-generation | Generate printable labels and identifiers |
| collection-checklist | Mobile-friendly collection checklist for collectors |
| chain-of-custody-log | Immutable log of custody transfers |
| sample-location-tracking | Track specimen location (fridge, courier, lab) |
