# Messaging Hub

Provides multi-channel message sending (SMS, Email, In-App) and routing, templates, provider adapters, delivery receipts, and message history.

Functions (Booking-level):
- `sms-delivery` — send and track SMS
- `email-delivery` — send and track Email
- `in-app-notifications` — send internal notifications
- `message-routing-and-delivery` — route messages to providers and handle webhooks
- `message-history-and-logging` — persistent history and audit trail
- `template-management` — manage templates and versions
