# Natural-Language Queries (Quick Reports)

Purpose
Allow managers to ask questions in natural language and receive concise, explainable reports.

Primary Users
- Managers, owners

Examples
- "Show top 5 delays today"; "Compare revenue last week vs. this week"

Acceptance Criteria
- Responses include data source, timestamp, and model version; allow drill-down to source data.

Implementation Notes
- Include rate limits and guardrails to prevent PHI leakage.
