# Owner Snapshot

Purpose
Provide the Owner role with a condensed, high-level view of pace, revenue, and key performance indicators for the selected horizon.

Primary Users
- Owners, senior managers

Key Metrics
- Chair utilization, throughput, completed procedures, realized revenue, AR aging snapshot
- Variance vs target, trending, and quick drill-down links

Acceptance Criteria
- Snapshot loads in under 2s and summarizes the day with links to deeper views.

Implementation Notes
- Respect cross-location scoping and allow owner to select clinics under their portfolio.
